### Feature
    - Login
    - Register
    - Verify account by firebase
    - Search user by name / by phone number
    - Chat:
        + Text, image file, video file, emoji
        + Recover message
    - Video call
    - Change info user
    - Add friend / block user
    - Create group, join group 
    - Manager group
    - Manager file in chat
